# Проект-приложение "Skybank" по анализу транзакций ^_^
## Проект представляет собой программное решение для работы с финансовыми данными, включая анализ трат, конвертацию валют, работу с акциями и другими финансовыми инструментами. Проект использует Python и различные библиотеки для обработки данных и взаимодействия с API.

## Описание: 
**В пакете src в модуле views.py реализована работа нашего приложения, а именно:**

1. Чтение данных из Excel.

2. Приветствие пользователя в зависимости от времени суток,
подсчет суммы трат по всем картам, 
подсчет кэшбэка по картам, 
вывод топ 5 транзакций за выбранный месяц.

3. Конвертация валют по актуальному курсу.

4. Получение данных о котировках 5-ти крупнейших акций S&P500.

5. Также в модуле services.py реализована функция для 
поисковой строки запроса от клиента, сортирует операции по ключевым данным(Описание или Категория).

6. А также в модуле reports.py реализованы функции для сортировки трат по категориям.
